# 枚举成员

## 定义
枚举成员分为常量枚举和计算枚举

## 常量枚举
满足一下条件之一时，他就是一个常量枚举表达式：
1. 一个枚举表达式字面量（主要是字符串字面量或数字字面量）
2. 一个对之前定义的常量枚举成员的引用（可以在不同的枚举类型中定义）
3. 带括号的常量枚举表达式
4. 一元运算符+，-， ～其中之一应用在了常量枚举表达式
5. 常量枚举表达式作为二元运算符+，-，*，/，%， <<，>>，>>>，&，｜，^的操作对象，若常数枚举表达式求值后为NaN或Infinity，则会在编译阶段报错。
```ts
// 上一节说到的就是常量枚举
enum Color1 = {
    green, 
    black,
    yellow,
}
```

## 计算枚举
例如：
```ts
enum Color1 = {
    green, 
    black,
    yellow = 'plm'.length,
}

```
